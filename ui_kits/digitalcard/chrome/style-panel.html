Customization model

Overrides are a partial that merges onto the active theme at render time. You can reset any field back to the theme default.

Accent is locked to indigo in chrome

The builder UI always uses the indigo system accent. What you set here applies only to your card's rendered output.

Contrast checker

Cards are user-customized, so the builder can't guarantee AAA. The live checker shows the current ratio and flags failures with a destructive badge — it guides without blocking.

Font scope

Fonts chosen here load on the public card page only. The builder chrome always uses Geist regardless.
