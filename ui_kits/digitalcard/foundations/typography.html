Note

Chrome font = Geist, not Inter. Inter is the monoculture default for SaaS products. Geist is Vercel's own typeface — neutral, precise, and slightly more characterful at display sizes. It keeps the chrome calm while staying in family with Vercel-stack aesthetics. Card fonts are entirely separate: each theme specifies its own Google Font family, loaded on demand. Geist never appears on a published card unless a theme explicitly chooses it. That separation is intentional — the chrome fades into the background, and the card's typography does the expressive work.
